## How to run

1. Clone this project
    
    ```
    git clone https://github.com/Techeer3-Spring-Study/TAB-Frontend.git
    ```
    
2. node module install
    
    ```yaml
    npm install (--legacy-peer-deps)
    ```
    
3. custom font link
    
    ```yaml
    npx react-native link
    ```
    
4. Android run
    
    ```
    npm run android
    ```
    
5. iOS run
    
    ```
    cd ios
    pod install
    cd ..
    npm run ios
    ```
